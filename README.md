# site-checker-service

Асинхронный сервис для мониторинга доступности сайтов. Включает в себя REST API для управления целями и независимый фоновый сканер

# Как запустить

 1. Клонируем репозиторий
git clone https://github.com/zstod/site-checker-service

 2. Создаем конфиг из шаблона (для локалхоста можно ничего не менять внутри)

cp .env.example .env

 4. Поднимаем проект

docker-compose up -d --build

После запуска API Swagger будут доступны по адресу: http://localhost:8000/docs




# Стек технологий
Backend: Python 3.14+, FastAPI, SQLModel

Database: PostgreSQL

Async: asyncio, httpx

Auth: JWT, bcrypt

Infra: Docker, Docker Compose

# Что реализовано
Регистрация и JWT-авторизация пользователей

CRUD-операции для добавления сайтов в систему мониторинга

Изолированный фоновый воркер, который по расписанию пингует сайты и сохраняет результаты (код ответа, время отклика, статус)

Автоматическая инициализация базы данных при старте

Настроены правильные зависимости в docker-compose
